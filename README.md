# notes-backend
Aplikasi backend menggunakan Hapi (http framework for NodeJS)

## How to run
- Clonning repo
```
git clone https://github.com/Indra2108/notes-backend.git
cd notes-backend
npm install
```

- Running server
```
npm run start
```

- Using Front-end
  - Visit http://notesapp-v1.dicodingacademy.com/
  - Select 'Change URL'
  - Input: http://localhost:3000/, press Enter.
